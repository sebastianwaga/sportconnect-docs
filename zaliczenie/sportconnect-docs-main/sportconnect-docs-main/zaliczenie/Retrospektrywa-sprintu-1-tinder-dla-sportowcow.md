## 3. Dyskusja i wnioski

**Główny problem do rozwiązania:**  
Niedokładnie opisane zadania na etapie planowania.

**Analiza (5 Whys):**  
Dlaczego zadania były niedokładnie opisane? → Bo nie zadano wystarczająco dużo pytań podczas planowania.  
Dlaczego nie zadano pytań? → Bo planowanie było zbyt szybkie.  
Dlaczego było szybkie? → Bo priorytetem było jak najszybsze rozpoczęcie implementacji.  
Dlaczego to było priorytetem? → Bo istniała obawa, że sprint będzie zbyt obciążony.  
Wniosek: Należy poświęcić więcej czasu na doprecyzowanie wymagań przed rozpoczęciem prac.
